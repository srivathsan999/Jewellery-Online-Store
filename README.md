# Jewellery Online Store - Premium HTML Template

A premium, multipurpose HTML template designed for selling jewellery online. Perfect for ThemeForest or TemplateMonster.

## Features

- **Luxury Design**: Elegant, premium, minimal design with high-trust aesthetics
- **Fully Responsive**: Mobile-first design that works on all devices
- **Dark Mode**: Toggle between light and dark themes
- **Smooth Animations**: GSAP and AOS animations throughout
- **Complete E-commerce**: Full shopping cart, wishlist, and checkout flow
- **24+ Demo Products**: Pre-loaded with sample jewellery products
- **User Dashboard**: Complete account management system
- **Admin Dashboard**: Full admin panel for managing the store
- **SEO Optimized**: Clean structure and meta tags
- **Performance Optimized**: Lazy loading and optimized assets

## Design System

- **Primary Color**: Deep Gold (#D4AF37)
- **Secondary Color**: Pearl White (#F8F8F4)
- **Accent Gradient**: Gold → Rose Gold
- **Typography**: Playfair Display (headings) + Inter (body)
- **Icons**: Feather Icons + Line Icons
